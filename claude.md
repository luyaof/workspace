# Workspace

本仓库专门用于托管静态网页，通过 **GitHub Pages** 进行展示。

## 访问地址

https://luyaof.github.io/workspace/

## 用途

- 静态网页展示
- 数据可视化 Dashboard
- 项目演示页面

## 技术栈

- HTML / CSS / JavaScript
- GitHub Pages（静态托管）

## ⚠️ 注意事项

**本仓库是公开的**，请确保：

- ❌ 不要上传任何敏感信息（API 密钥、密码、个人隐私数据等）
- ❌ 不要提交 `.env` 文件或任何包含凭证的配置文件
- ✅ 仅上传可公开展示的静态内容
